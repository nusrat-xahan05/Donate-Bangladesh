Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web. DOM allows JavaScript to interact with and manipulate the structure, content, and style of web pages. Such as: DOM allows JavaScript to change all the HTML elements, HTML attributes & all the CSS styles in the page, remove existing HTML elements and attributes, dd new HTML elements and attributes, react to all existing HTML events in the page

How do you select an element from the DOM?

Followings are the methods to select an element from the DOM:
    1. document.getElementsByTagName()
    2. document.getElementById()
    3. document.getElementsByClassName()
    4. document.querySelectorAll()
    5. document.querySelector()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the pattern of the DOM, where a single event listener is added to a parent element to manage events triggered by its child elements.
Event Delegation improves the performance, simplifies the code by reducing repeatation & handle the content dynamically.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes using the DOM, there are mainly two methods:
    1. setAttribute(): Use element.setAttribute('attributeName', 'value') to set or update an attribute
    2. getAttribute(): Use element.getAttribute('attributeName') to retrieve the value from that attribute.
Sometimes element.removeAttribute('attributeName') is used to remove an attribute.

To manipulate an element's styles using the DOM, there are also two methods:
    1. propertyName: element.style.propertyName = 'value' is used to change a specific CSS property
    2. className: element.className is used to change its styles based on predefined CSS classes. ClassList property can also use which provides methods like add(), remove(), and toggle(). Such as, element.classList.add(className).
